# Курсовая работа по дисциплине "Параллельное, распределенное и многоязыковое программирование"

Цель работы – разработать с использованием C++ динамически подключаемую библиотеку для работы с матрицами и графический интерфейс пользователя с использованием Python.

Более подробно задачи и некоторые метрики приведены в odt-файле. 

### Компоновка динамической библиотеки:

```g++ -O3 -march=native -fipo-pta -fopenmp -shared -std=c++20 -fPIC $(python3 -m pybind11 --includes) pybind_wrap.cpp  Matrix.cpp -o Matrix.so```
